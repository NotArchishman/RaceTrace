# Formula 1 Live Dashboard

A Streamlit-based interactive Formula 1 analytics app powered by **FastF1**.
Displays live and historical race data, driver comparisons, and circuit maps.

## Features
- Real-time race analytics using FastF1
- Driver performance comparison
- Lap-by-lap trend visualization
- Circuit and team overview
- Dynamic updates via Streamlit UI
- Pit Stop visualization and statistics
- Tyre compound usage analytics

## Run Instructions
```bash
pip install -r requirements.txt
streamlit run Race_Trace.py
